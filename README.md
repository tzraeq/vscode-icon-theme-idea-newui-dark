# JetBrains New UI Dark Icons

This extension provides a VS Code **file icon theme** inspired by JetBrains IDE New UI Dark style.
It focuses on Java and Maven projects while keeping a clean, consistent look for common files and folders.

## Screenshot

![JetBrains New UI Dark Icons](screenshots/intellij-dark.png "JetBrains New UI Dark Icons")

## Features

- JetBrains New UI Dark style icons for files and folders
- Tuned for Java ecosystem workflows
  - Java source and resources
  - Maven structure and key files
- Clean dark-friendly icon contrast for side bar readability

## Install

- Install from [Visual Studio Marketplace][marketplace]

## Usage

1. Open VS Code Command Palette (`Ctrl` + `Shift` + `P`)
2. Run `Preferences: File Icon Theme`
3. Select `IDEA New UI Dark Icons`
